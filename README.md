# Tour Website
**Tour Website** is an elegant and responsive web project built using **HTML5, CSS3, and JavaScript**. Designed to showcase beautiful travel destinations and tour packages, this site provides users with a modern, visually appealing browsing experience.

# Features
- **Responsive Design** — Looks great on mobile, tablet, and desktop devices
- **Tour Showcase** — Displays tour information such as images, prices, and details
- **Clean UI** — Minimal and structured layout for easy navigation
- **Interactive Elements** — Smooth button actions and transitions
- **Pure HTML, CSS & JS** — No frameworks, just core web technologies

# Technologies Used
- **HTML5** – for semantic structure and layout
- **CSS3** – for styling and responsive design
- **JavaScript** – for interactivity and dynamic UI behavior
